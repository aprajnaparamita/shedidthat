## Usage Examples

**Chat Screen:**
- Background: `#1A0522`
- Jess bubbles: `#F5F5F5` with `#2D0838` text
- User bubbles: `#E91E8C` with `#FFFFFF` text
- Timestamps: `#8B6B96`

**Input Bar:**
- Background: `#3D1048`
- Border (inactive): `#5A3F63`
- Border (focused): `#E91E8C`
- Text: `#FFFFFF`
- Placeholder: `#8B6B96`

- Background: `#2D0838`
- Conversation cards: `#3D1048`
- Card text: `#FFFFFF`
- Preview text: `#C9A3D6`
- Time: `#8B6B96`

**Buttons:**
- Primary (Start Debrief): `#E91E8C` bg, `#FFFFFF` text
- Secondary (Cancel): `#5A3F63` bg, `#FFFFFF` text
- Mic active: `#FF4757` pulsing
